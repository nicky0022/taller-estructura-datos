# Taller 1 - Clase Time en C#

Este proyecto implementa una clase llamada Time que permite trabajar con horas, minutos, segundos y milisegundos.

## Funcionalidades

- Constructores sobrecargados
- Conversión a milisegundos, segundos y minutos
- Suma de tiempos
- Verificación si pasa al siguiente día
- Formato de hora no militar

## Cómo ejecutar

dotnet run desde la terminal
